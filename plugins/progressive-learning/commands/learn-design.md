# Learn Design

Invoke the learn-design skill to architect a progressive learning curriculum.

**If no curriculum exists:** Starts the new-area flow -- asks about background, goals, interests, and designs a project sequence.

**If curriculum exists:** Runs a retrospective on the completed project, updates the learner profile, and generates the next project's phase map.

Check for `docs/curriculum.md` in the current working directory to determine mode.
