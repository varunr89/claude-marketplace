# Learn

Invoke the learn skill to start or continue a learning session.

Reads the active project's phase map, scans code state, checks conversation history, and proposes where to pick up. Then guides the learner through interleaved learn-build spirals using Socratic teaching.

If no phase map exists, tell the learner to run `/learn-design` first.
